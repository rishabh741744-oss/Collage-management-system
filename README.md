## ✨ Features
- **Real-time Firebase Database** - Data instantly syncs without refresh
- **Role Based Login** - Separate Admin & Student access
- **Student Management** - Add, Edit, Delete, Search students
- **Weekly Attendance Board** - Track 7-day presence with date picker
- **PDF Export** - Download complete student list
- **Mobile Responsive** - Works perfectly on phone + desktop
- **Glassmorphism UI** - Modern beautiful design

## 📊 Dashboard Overview
- Total Students Count
- Present Today Count  
- Average Attendance %
- Weekly Attendance Tracker with Present/Absent markers

## 🛠️ Tech Stack
HTML5, Tailwind CSS, JavaScript, Firebase Realtime Database
